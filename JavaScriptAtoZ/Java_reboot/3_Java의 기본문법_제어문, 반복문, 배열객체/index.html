배열 객체

[] 를 이용해 하나의 변수로 여러 개의 변수를 표현 하는 것이다. 자바에서는 객체이다. 따라서 row의 길이가 각각 다 다른, 다차원 배열을 만들 수도 있고,각각이 객체이기 때문에 배열에 타입이 다른 값을 저장 할 수 있다. 

int arr1[];
int[] arr2;
//둘다 가능하다.

[]내에서는 첨자 사용을 지정하지 않아도 되나, 배열 선언만으로는 배열이 메모리에 할당 되지 않는다. 따라서 직접 const한 값을 지정해주거나, new를 이용해 메모리 공간을 할당 해 주어야 한다. 

int arr1[]={1,2,3,4};
int[] arr2={1,2,3,4};
//둘다 가능하다.

new 를 이용할 경우 다음과 같이 해야한다. 

int arr1[]=new int[10];
int []arr2=new int[10];
int arr3[];
arr3=new int[10];
